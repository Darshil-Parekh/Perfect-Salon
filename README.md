# 💇 Perfect Hair & Beauty Salon Website

A modern, responsive salon website built using **HTML, CSS, Bootstrap, and Animate.css**.
This project showcases a stylish landing page for a beauty salon including services, gallery, reviews, and contact information.

---

# 🌐 Live Preview

https://darshil-parekh.github.io/Perfect-Salon/

---

# ✨ Features

* Responsive design (Desktop, Tablet, Mobile)
* Modern glassmorphism UI
* Animated hero section
* Service cards with hover effects
* Image gallery section
* Google review integration
* Contact and footer section
* Smooth layout using Bootstrap Grid
* Animate.css for entrance animations

---

# 🛠️ Built With

* **HTML5**
* **CSS3**
* **Bootstrap 5**
* **Animate.css**
* **Google Fonts (Poppins)**
